# 1. 设计理念（Design Principles）

## 1.1 品牌定位

智慧农业生态园官网定位为现代农业产业园官方网站。

整体视觉应体现：

- 专业（Professional）
- 简约（Minimal）
- 可信（Reliable）
- 科技（Technology）
- 自然（Nature）

整体风格参考日本制造业及农业科技企业官网。

参考品牌：

- Kubota
- Yanmar
- Panasonic
- Mitsubishi Electric
- Alps Alpine

页面不追求互联网产品的炫酷效果，而强调企业品牌形象与信息传达。

---

## 1.2 设计原则

整个网站遵循以下原则：

### Less is More

减少装饰元素。

保持页面整洁。

### Content First

突出内容。

设计服务于内容，而非喧宾夺主。

### Consistency

统一设计语言。

所有页面保持一致的组件、颜色、间距及交互方式。

### Whitespace

大量留白。

通过留白建立视觉层级，而不是依赖复杂装饰。

### Professional

商务感优先。

体现企业可信度，而非营销感。
